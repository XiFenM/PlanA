Success in this role will require deep knowledge of Data Center, Client, Endpoint AI workloads such as LLM, Generative AI, Recommendation, and/or transformer … AI cross cloud, client, edge… the candidate needs to have hands-on experiences with various AI models, end-to-end pipeline, industry framework (pytrouch, vLLM, SGLang, llm-d,Triton) / SDKs and solutions.

胜任此职位需要对数据中心、客户端、端点AI工作负载（如大语言模型（LLM）、生成式AI、推荐系统和/或变压器……AI跨云、客户端、边缘……）有深入了解，候选人需要具备使用各种AI模型、端到端管道、行业框架（pytrouch、vLLM、SGLang、llm-d、Triton）/软件开发工具包（SDK）和解决方案的实践经验。

**主要职责：**

- Position technical proposals / enablement to (blogs, tutorials, user guide…) AI SW developers and/or top customers.
- 向（博客、教程、用户指南…）AI软件开发者和/或顶级客户定位技术提案/赋能。

- 为AI软件开发者/社区和/或客户的概念验证（PoC）成功做出重大贡献。

- 推动AI开发者/社区/客户对AI软件、解决方案路线图规划的需求。

- 分析竞争解决方案，以确定其优势和劣势，从而阐明AMD AI软件和解决方案的价值主张。
    
- Provide inputs / feedback to AI SW / hardware silicon / board roadmap for AI cross cloud, client, and edge...
    
- 为AI跨云、客户端和边缘的AI软件/硬件芯片/电路板路线图提供输入/反馈...
    

**PREFERRED EXPERIENCE:**

**优先考虑的经验：**

- With experience to use LLMs or design an Agent to generate GPU kernels code (CUDA, CUTLASS, Triton, HIP) or other kernels code; With experience to use LLMs or design an Agent pipeline for some domain specific or vertical applications
    
- 有使用大语言模型（LLMs）或设计智能体（Agent）来生成GPU内核代码（CUDA、CUTLASS、Triton、HIP）或其他内核代码的经验；有使用大语言模型（LLMs）或设计智能体（Agent）流水线以用于特定领域或垂直应用的经验

- 具备扎实的沟通技巧，能够有效阐述架构提案和价值主张。
